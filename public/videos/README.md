# NuFounders Hero Videos

Place your b-roll videos in this directory for the landing page hero carousel.

## Expected File Names

The video carousel will automatically detect and use videos with these names:

- `hero-1.mp4`
- `hero-2.mp4`
- `hero-3.mp4`
- `hero-4.mp4`
- `hero-5.mp4`
- `entrepreneur-1.mp4`
- `entrepreneur-2.mp4`
- `success-1.mp4`
- `success-2.mp4`
- `community-1.mp4`

## Video Specifications

- **Format:** MP4 (H.264 codec recommended)
- **Resolution:** 1920x1080 (1080p) or 3840x2160 (4K)
- **Duration:** 10-30 seconds per clip
- **Aspect Ratio:** 16:9 (landscape)
- **File Size:** Keep under 20MB per video for optimal loading

## Recommended Content

Based on `VIDEO_PROMPTS.md`, prioritize these types of clips:

1. **Entrepreneurship scenes** - Business success moments, confident leaders
2. **Learning moments** - Course completion, studying, achievements
3. **Community connections** - Networking, mentorship, group celebrations
4. **Lifestyle/Inspiration** - Morning routines, work-life balance, vision

## Performance Tips

- Compress videos using tools like HandBrake or FFmpeg
- Consider using WebM format for additional browser optimization
- The carousel preloads upcoming videos for smooth transitions

## Fallback Behavior

If no videos are present, the hero section displays an animated gradient background.
